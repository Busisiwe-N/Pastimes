## 🌟 Project Overview

Pastimes is a professional online clothing marketplace designed to connect buyers and sellers through a secure and user-friendly platform.

The application allows users to:

✔ Register and create an account  
✔ Login securely using hashed passwords  
✔ Browse clothing items by category  
✔ Add products to a shopping cart  
✔ Upload clothing items for sale  
✔ Communicate through messages  
✔ Manage orders and purchases  
✔ Receive administrator verification  
✔ Shop clothing for Men, Women, Teenagers, and Kids

---

## 🎯 Project Objectives

The purpose of Pastimes is to:

- Promote sustainable fashion.
- Encourage clothing reuse and resale.
- Provide a secure marketplace for users.
- Deliver a visually appealing shopping experience.
- Demonstrate full-stack web development skills.
